# Speech-To-Text and Text + Video Recommendation AI Mobile App

## Description
  This project is a mobile application that allows users to convert speech to text and receive video recommendations based on the transcribed text. 
  The app uses the Speech-to-Text API for transcription and the Groq AI model for generating video recommendations from YouTube. The backend is built with Express.js, and the app is developed using React Native with Expo.

## Technologies
  - Expo App
  - Express.js
  - Speech-to-Text API
  - Groq AI
  - YouTube Data API
